# Web Penetration Tester Notes

This repo contains information and notes about my preparation for the CWES certification, based on the HTB Web Penetration Tester path and Burp Suite labs.

The source material is entirely from HTB and Burp Suite — I take those sessions and write my own Cheatsheets covering what I consider important or worth highlighting.

I will also upload my personal Notes over time, but for now I'm uploading my Cheatsheets along with the full module content I've studied. The reason for keeping both is so that if my Cheatsheet is missing something, I can go back to the `.md` files, reread the section, and figure out what I missed or what I need to study more deeply.

## Structure

- **Cheatsheets** — condensed reference material I put together from each module.
- **Module notes** — fuller content from each studied module, used as a fallback when the cheatsheet falls short.
